## Il problema

Lavori per più clienti a pacchetti di ore. Salti da un progetto all'altro decine di volte al giorno.
A fine mese arriva sempre la stessa domanda: *quante ore ho fatto davvero per ciascuno?*

Fogli Excel dimenticati, timer lasciati accesi tutta la notte, ore stimate "a occhio", fatture compilate a mano.
Tempo perso, soldi persi, e quel dubbio fastidioso di non aver contato tutto.

## La soluzione

**TimeTracker vive nella tray** e conta le ore per te, senza interromperti.

- ⌨️ Premi `Ctrl/⌘ + Alt + lettera` da **qualsiasi programma** → parte il timer di quel cliente. Un solo timer alla volta, niente doppi conteggi.
- 💤 Ti allontani dal PC? Dopo qualche minuto di inattività **si ferma da solo** e retrodata lo stop all'ultimo momento attivo. Niente più nottate fatturate per sbaglio.
- 📸 Mentre lavori scatta **screenshot di prova** del progetto attivo, salvati in locale: prova del lavoro svolto, se il cliente la chiede.
- 📊 A fine settimana o mese, un clic e hai il **report** per cliente; un altro clic e hai la **fattura PDF** con numerazione progressiva, imponibile, IVA e totale.

Nessun foglio di calcolo. Nessun timer dimenticato. Solo ore reali, pronte da fatturare.

## ✨ Cosa lo rende diverso

- ⚡ **Hotkey globali** — avvii e fermi i clienti senza nemmeno aprire la finestra. `Ctrl/⌘ + Alt + 0` ferma tutto, `+ T` riapre l'app.
- 💤 **Auto-pausa intelligente** — il timer si ferma da solo quando sei inattivo e corregge l'orario di stop. Le ore restano oneste.
- 📸 **Screenshot automatici** — intervallo configurabile, salvati in una cartella locale, mai inviati da nessuna parte.
- 🧾 **Fatture PDF pronte** — dati emittente e cliente, tariffa €/h, imponibile, IVA, IBAN e note. Numerazione automatica.
- 📊 **Report flessibili** — settimana, mese, settimana/mese scorsi o intervallo personalizzato. Export **CSV** (Excel) e **PDF**.
- 🌍 **12 lingue in automatico** — l'app parla la lingua del tuo sistema appena la apri, e la cambi quando vuoi.
- 📱 **Interfaccia adattiva** — layout pulito che si ridimensiona su finestre piccole e grandi.
- 🛡️ **Niente porte aperte, niente backdoor, niente telemetria** — zero connessioni di rete, dati solo sul tuo computer.
- 🪶 **Leggera** — resta in background nella tray e non ti sta tra i piedi.

## 🔒 Privacy & sicurezza

TimeTracker è costruito per **non fidarsi di nulla** e non far uscire niente.

- 🚫 **Zero rete** — l'app non si connette a internet. Nessuna telemetria, nessun account, nessun cloud.
- 💾 **Dati 100% locali** — ore, clienti e screenshot restano solo sul tuo computer.
- 🛡️ **Blindata** — `contextIsolation`, `sandbox`, navigazione esterna bloccata e **Content-Security-Policy** severa. Nessuna "porta aperta" da attaccare.
- 📖 **Codice leggibile** — niente dipendenze oscure: l'unica libreria a runtime serve solo per gli screenshot.

Dove sono i dati:
- **Windows:** `%APPDATA%/TimeTracker`
- **macOS:** `~/Library/Application Support/TimeTracker`

> Le fatture PDF sono **copie di cortesia**: non sostituiscono la fattura elettronica XML (FatturaPA / SdI) richiesta in Italia.

## 🛠️ Compilare da sorgente

Richiede [Node.js](https://nodejs.org) 18+.

```bash
git clone https://github.com/blackstardigitalstudio/time-tracker.git
cd time-tracker
npm install
npm start        # avvia in sviluppo
npm run dist     # crea installer + portable in ./dist
```

**Pubblicare le release (Windows + macOS automatici):** fai push e crea un tag di versione — il workflow [GitHub Actions](.github/workflows/build.yml) compila tutto nel cloud (senza bisogno di un Mac) e allega `.exe`, `.zip` e `.dmg` alla Release.

```bash
git tag v1.2.0 && git push origin v1.2.0
```

> 💡 Su Windows, la creazione dell'installer in locale richiede una volta la **Modalità sviluppatore** (Impostazioni → Privacy e sicurezza → Per sviluppatori) oppure un terminale da amministratore. La **versione portatile** invece si genera senza alcun permesso speciale.

## 🇬🇧 English

**TimeTracker** is a time tracker for **freelancers** juggling hourly packages across multiple clients.

Switch client with a **global hotkey** from any app, let it take **proof screenshots**, have it **auto-pause** when you step away, and generate **reports** and **PDF invoices** at month's end. **12 languages**, **Windows & macOS**.

- ⌨️ **Global hotkeys** — `Ctrl/⌘ + Alt + letter` to start/stop a client from anywhere. One timer at a time.
- 💤 **Smart auto-pause** — stops by itself when you're idle and back-dates the stop time.
- 📸 **Automatic screenshots** of the active project, saved locally (never uploaded).
- 📊 **Reports** by week/month/custom range, with **CSV** and **PDF** export.
- 🧾 **PDF invoices** with progressive numbering, issuer/client details, net, VAT and total.
- 🌍 **12 languages**, auto-detected from your system and switchable anytime — all offline.
- 🛡️ **No open ports, no backdoors, no telemetry** — zero network, data stays on your machine.
